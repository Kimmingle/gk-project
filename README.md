# 🖥 사내 관리 시스템 (React)

사내 사용자 및 게시판 관리를 위한 **React 기반 웹 애플리케이션**입니다.  
로그인/회원가입부터 관리자 기능, 권한 관리, 게시판 CRUD까지  
사내 운영에 필요한 기본 기능들을 포함하고 있습니다.

---

## 📌 프로젝트 개요

- **프로젝트 목적**
  - 사내 사용자 관리 및 게시판 운영 자동화
  - 권한 기반 기능 분리로 보안 강화
  - 유지보수 및 확장성을 고려한 프론트엔드 구조 설계

- **대상 사용자**
  - 일반 사용자
  - 관리자(Admin)

---

## 🛠 기술 스택

- **Frontend**
  - React
  - JavaScript (ES6+)
  - Axios
  - React Router

- **상태 관리**
  - Context API / Redux (사용 시 명시)

- **스타일링**
  - CSS / SCSS / Styled-components (사용한 것 명시)

- **기타**
  - Git / GitHub
  - REST API 연동

---

## ✨ 주요 기능

### 🔐 인증 / 인가
- 로그인
- 회원가입
- 로그아웃
- 토큰 기반 인증 처리
- 권한(Role) 기반 접근 제어

---

### 👤 회원 관리
- 회원 정보 조회
- 회원 정보 수정
- 회원 권한 관리 (관리자 전용)

---

### 🛡 관리자 기능
- 전체 회원 목록 조회
- 회원 권한 변경
- 게시글 관리 (삭제 / 수정)
- 관리자 전용 페이지 접근 제한

---

### 📝 게시판
- 게시글 목록 조회
- 게시글 상세 조회
- 게시글 작성
- 게시글 수정
- 게시글 삭제
- 작성자/권한에 따른 기능 제한

---

### 🔄 CRUD 기능
- 사용자 CRUD
- 게시글 CRUD
- 관리자 권한에 따른 CRUD 제한

---

### 🔑 권한 관리
- 사용자 / 관리자 권한 분리
- 권한별 UI 및 API 접근 제어
- 관리자 전용 메뉴 노출 제어

---
